# STM32-Nucleo-411RE-Bare-Metal-Digital-Thermostat-Display
## About
Bare metal driver for integrating and displaying temperature and humidity data between STM32 Nucleo 411RE MCU, DHT 11 temperature and humidity sensor, and SSD1306 Oled display screen.

## Tools/Components
* STM32 Nucleo 411RE development board
* DHT11 Temperature and Humidity Sensor
* SSD1306 OLED display
* Segger Ozone
* Unity - unit test framework
